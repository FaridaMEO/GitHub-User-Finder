# GitHub User Finder

**Автор:** [Османов Али]

## Описание

Приложение для поиска пользователей GitHub с возможностью сохранения избранных. Использует официальный GitHub API для получения данных о пользователях.

## Функциональность

* Поиск пользователей GitHub по имени.
* Отображение результатов в табличном виде (имя пользователя, полное имя, местоположение).
* Добавление пользователей в избранное.
* Сохранение избранных пользователей в JSON‑файл.
* Просмотр списка избранных пользователей.

## Использование API

Приложение использует GitHub API v3:

* **Endpoint:** `https://api.github.com/users/{username}`
* **Метод:** GET
* **Ограничения:** Без аутентификации — 60 запросов в час. С аутентификацией — 5000 запросов в час.

Для повышения лимита запросов можно добавить аутентификацию через Personal Access Token:

1. Создайте токен в настройках GitHub (Settings → Developer settings → Personal access tokens).
2. Добавьте заголовок авторизации в запросы:
   ```python
   headers = {"Authorization": "token YOUR_TOKEN"}
