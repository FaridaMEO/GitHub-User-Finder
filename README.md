
# GitHub User Finder

**Автор:** [Ваше Имя Фамилия]

## Описание

Приложение для поиска пользователей GitHub с возможностью сохранения избранных пользователей.

## Функциональность

* Поиск пользователей GitHub по имени.
* Отображение результатов поиска в виде списка.
* Добавление пользователей в избранное.
* Сохранение избранных пользователей в JSON‑файл.

## Использование API GitHub

Приложение использует GitHub API v3:
* **Endpoint:** `https://api.github.com/search/users?q={query}`
* **Метод:** GET
* **Параметры:**
  * `q` — поисковый запрос (имя пользователя).

API позволяет выполнять до 10 запросов в минуту для неавторизованных пользователей.

## Установка и запуск

1. Установите зависимости: `pip install requests`.
2. Запустите приложение: `python main.py`.

## Примеры использования

1. Введите имя пользователя в поле поиска и нажмите «Поиск».
2. Выберите пользователя из списка результатов.
3. Нажмите «Добавить в избранное», чтобы сохранить пользователя.
4. Избранные пользователи сохраняются в `favorites.json`.
